# Memory Lock Primitives

![Development Status](https://img.shields.io/badge/status-active--development-blue.svg)

Page-locking vocabulary — `Memory.Lock` namespace with `Kind`, `Token`, and `Error` types.

Sibling extraction of [swift-memory-primitives](https://github.com/swift-primitives/swift-memory-primitives). Syscall implementations live in platform packages — POSIX in [swift-iso-9945](https://github.com/swift-standards/swift-iso-9945) (`extension Memory.Lock`), Windows in [swift-windows-standard](https://github.com/swift-standards/swift-windows-standard).
